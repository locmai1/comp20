Describe the API you selected and what it does

This uses the icanhazdadjoke API which allows users to fetch a variety of dad jokes!

Cite the website where you found it

https://icanhazdadjoke.com/api#calling-the-api

Describe the options/parameters you used for the API request

I used the term parameters which when set, requests all dad jokes with that term. This is represented above by the keyword input and makes an API request to the API with the query string: ?term=keyword
